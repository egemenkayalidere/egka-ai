# Web App

Next.js web uygulaması.

## 🚀 Teknolojiler

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Query

## 🌐 Özellikler

- Modern web uygulaması
- SEO optimizasyonu
- Server-side rendering
- Responsive design
- Progressive Web App (PWA)

## 🛠️ Kurulum

```bash
npm install
npm run dev
```

## 📁 Proje Yapısı

```
web-app/
├── src/
│   ├── app/
│   ├── components/
│   ├── lib/
│   ├── hooks/
│   ├── types/
│   └── styles/
├── public/
├── next.config.js
└── package.json
```

## 🔧 Özellikler

- **App Router**: Next.js 13+ App Router
- **TypeScript**: Tam tip güvenliği
- **Tailwind CSS**: Modern CSS framework
- **Framer Motion**: Animasyonlar
- **React Query**: Veri yönetimi
