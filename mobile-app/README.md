# Mobile App

React Native/Expo mobile uygulaması.

## 🚀 Teknolojiler

- React Native
- Expo
- TypeScript
- React Navigation
- AsyncStorage

## 📱 Özellikler

- Cross-platform (iOS & Android)
- Modern UI/UX
- Offline support
- Push notifications

## 🛠️ Kurulum

```bash
npm install
npx expo start
```
