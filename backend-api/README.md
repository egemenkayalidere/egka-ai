# Backend API

Node.js/Express API server.

## 🚀 Teknolojiler

- Node.js
- Express
- TypeScript
- MongoDB
- Firebase
- JWT Authentication

## 🔧 Özellikler

- RESTful API
- Authentication & Authorization
- Database integration
- File upload
- Real-time features
- API documentation

## 🛠️ Kurulum

```bash
npm install
npm run dev
```

## 📁 Proje Yapısı

```
backend-api/
├── src/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   ├── middleware/
│   ├── services/
│   ├── utils/
│   └── types/
├── config/
└── package.json
```

## 🔌 API Endpoints

- `GET /api/health` - Health check
- `POST /api/auth/login` - User login
- `POST /api/auth/register` - User registration
- `GET /api/users` - Get users
- `POST /api/users` - Create user
