# Admin Panel

React/TypeScript admin dashboard uygulaması.

## 🚀 Teknolojiler

- React
- TypeScript
- Next.js
- Tailwind CSS
- React Query
- Zustand

## 📊 Özellikler

- Modern dashboard UI
- Real-time data
- User management
- Analytics
- Responsive design

## 🛠️ Kurulum

```bash
npm install
npm run dev
```
